## Install all bundles

```
git submodule update --init --recursive
```

## Update all bundles

```
git submodule update --recursive
```

## Add a bundle

```
cd ~/.vim
git submodule add REPOSITORY bundle/PLUGIN-NAME
```

## Install ctags

Depends on Unix installation

```
pacman -S ctags
brew install ctags
```
